# 📈 CryptoInsight AI

CryptoInsight AI is a production-grade, multi-agent cryptocurrency analysis dashboard built with Streamlit, LangGraph, and Groq (Llama-3). 

## Features
- **Multi-Agent Orchestration:** Data Fetcher, News Researcher, Technical Analyst, and Report Generator managed by a Supervisor.
- **Real-Time Data:** Pulls live price action via Alpha Vantage and breaking news via Tavily.
- **Interactive UI:** Professional dark-themed Streamlit frontend with Plotly charts.
- **Export Capabilities:** One-click PDF report generation.
- **Bilingual:** Generates reports in English or Urdu.

## Installation
1. Clone the repo and navigate to `CryptoInsight-AI`
2. `pip install -r requirements.txt`
3. Copy `.env.example` to `.env` and add your API keys.
4. Run `streamlit run app.py`mlit run app.py`
